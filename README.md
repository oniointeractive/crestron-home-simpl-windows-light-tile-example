# Crestron Home SIMPL Windows Light Tile Integration Example

This repository contains an example SIMPL Windows project for integrating Crestron Home with SIMPL Windows using the ONIO Crestron Home Light Tiles driver.

The example demonstrates how to connect Crestron Home Light Tile components, native Light Scenes, Quick Actions, and general room controls to a SIMPL Windows program.

## What This Example Covers

This project shows how to integrate:

- Crestron Home Light Tiles
- Crestron Home Quick Actions
- Crestron Home native Light Scenes
- General Crestron Home room controls
- Two-way communication between Crestron Home and SIMPL Windows

The goal of this example is to help Crestron programmers and AV integrators quickly understand how to connect Crestron Home functionality to a traditional SIMPL Windows program.

## Requirements

To use this example, you will need:

- A Crestron Home processor
- A SIMPL Windows processor
- Crestron Home Integration Modules
- A generated Crestron Home Integration Report
- The ONIO Crestron Home Light Tiles driver
- SIMPL Windows

## Crestron Home Integration Report

Before using this example, you need to generate the Integration Report from the myCrestron Residence portal.

The Integration Report contains the required IDs for rooms, functions, scenes, and actions. These IDs are used inside SIMPL Windows to connect the correct Crestron Home functions.

Crestron documentation:

https://docs.crestron.com/en-us/8525/Content/SIMPL%20Home%20Integration%20Modules.htm

## How to Use This Example

1. Add the ONIO Light Tiles driver to your Crestron Home project.
2. Add the required Light Tile components, Light Scenes, and Quick Actions.
3. Generate the Integration Report from myCrestron Residence.
4. Open the SIMPL Windows example project.
5. Enter the required room, function, scene, and action IDs from the Integration Report.
6. Compile and upload the programs to both processors.
7. Trigger the connection from the Crestron Module.
8. Test two-way communication between Crestron Home and SIMPL Windows.
